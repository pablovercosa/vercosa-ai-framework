## Objetivo

Esta pasta reúne a documentação inicial de conduta, convivência e colaboração do Vercosa AI Framework.

## O Que Esta Pasta Faz

- Aponta para o [código de conduta inicial](../../CODE_OF_CONDUCT.md).
- Mantém diretrizes práticas em [community-guidelines.md](community-guidelines.md).
- Relaciona conduta com contribuição, segurança e uso responsável.
- Registra limites do estágio atual sem prometer governança comunitária madura.

## O Que Esta Pasta Não Faz

- Não substitui o [guia de contribuição](../../CONTRIBUTING.md).
- Não substitui a [política inicial de segurança](../../SECURITY.md).
- Não define comitê formal, SLA, mediação formal ou processo disciplinar maduro.
- Não cria canal público definitivo para problemas de conduta.

Entradas:

- dúvidas de convivência;

Saídas:

- limites claros sobre governança comunitária ainda não madura.
