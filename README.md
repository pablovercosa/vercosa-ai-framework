# Vercosa AI Framework

An Open Source AI Specification-Driven Software Development Framework.

## Objetivo

O Vercosa AI Framework organiza o desenvolvimento de software assistido por IA usando especificações, missões, agentes, workflows, policies, knowledge hub e adapters.

## Estado

Projeto em fase de fundação arquitetural.

## Runtime inicial

O OpenCode será usado inicialmente como runtime/laboratório.

## Princípios

- Specification First
- AI Native
- Provider Agnostic
- Local First
- Extensible by Design
- Security by Design
- Token Efficiency
- Governance by Design

## Estrutura

- AGENTS.md: contexto central para agentes.
- knowledge/: decisões, visão, princípios e arquitetura.
- specs/: especificações.
- .opencode/: integração inicial com OpenCode.
- docs/: documentação.
- scripts/: automações futuras.
