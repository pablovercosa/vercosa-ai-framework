# Guardian Spec 004 — Cost Optimization

## Objetivo

Controlar custo de uso de modelos, ferramentas e infraestrutura.

## Regras

1. Usar modelos gratuitos ou locais para tarefas simples.
2. Usar modelos pagos somente quando necessário ou autorizado por política.
3. Preferir fallback gratuito quando não houver credencial paga.
4. Registrar uso de modelos quando possível.
5. Evitar chamadas repetidas ao mesmo contexto.
6. Reaproveitar resultados cacheados.
7. Permitir perfis: economy, balanced, premium.
8. O usuário pode definir orçamento por missão, projeto ou período.
